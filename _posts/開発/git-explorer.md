## アルファベットの意味（Git Status）

| 記号 | 意味 | 説明 |
|------|------|------|
| A | Added | 新規追加されたファイル（ステージ済み） |
| M | Modified | 変更されたファイル |
| U | Untracked | Gitで追跡されていない新規ファイル |
| D | Deleted | 削除されたファイル |
| C | Conflict | マージコンフリクト発生中 |
| R | Renamed | リネームされたファイル |
| S | Submodule | Gitサブモジュール |
| I | Ignored | .gitignoreで無視されているファイル |
| T | Type changed | ファイルタイプが変更（例: 通常ファイル→シンボリックリンク） |

## 数字の意味

| 記号 | 意味 |
|------|------|
| ①②... | ファイル/フォルダ内の変更ファイル数 |
